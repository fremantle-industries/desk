# Desk

An advisor reference library for [tai](https://github.com/fremantle-industries/tai)

## Installation

Add `desk` to your list of dependencies in `mix.exs`

```elixir
def deps do
  [
    {:desk, "~> 0.0.1"}
  ]
end
```
